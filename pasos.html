Pasos para trabajar con git

Cuando ya existe un proyecto creado y te lo han compartido copias el enlace y donde quieras dejar la carpeta del proyecto con click derecho pulsas en Git bash here. Se abrirá la consola de comandos y debes introducir el comando git clone y el enlace del proyecto. El proyecto se descargará y debes entrar en la carpeta creada y volver a hacer click derecho y git bash here. y, si quieres, hacer git pull para actualizar, aunque se entiende que está actualizada. Creas una rama nueva con checkout -b "nombre-rama" y trabajas en esa rama (lo puedes comprobar en la esquina inferior izquierda del Visual). Cuando termines haces git add . y git commit - m "descripción" y ya puedes hacer git push y copiar la línea de comando que te indica. Una vez hecho esto vas a Github y haces new pull request y esperas a que tus compañeros comenten y aprueben para poder hacer merge.
