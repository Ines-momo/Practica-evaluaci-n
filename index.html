Git es muy útil para trabajos colectivos, cuando necesitas colaborar con otras personas y necesitáis compartir vuestros trabajos. También sirve para que nunca se pierdan las versiones anteriores y puedas volver siempre a la base, si se necesitase por algún error.

Ventajas de usar git

  • Se puede trabajar con personas desde distintas ubicaciones
  • Puedes volver a versiones anteriores en caso de errores
  • Puedes trabajar desde distintos ordenadores de manera sencilla, solo necesitas clonar el trabajo desde git

Principales comandos git

Comandos Uso
git init Para iniciar el repositorio
git checkout -b "nombre-rama" Crea una nueva rama de proyecto
git checkout "nombre-rama" Cambia a una rama existente
